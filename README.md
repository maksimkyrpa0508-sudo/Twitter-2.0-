# 🐦 Twitter Clone (Fullstack Web App)

## 📌 Description
This project is a fullstack social media web application inspired by Twitter.

Users can create profiles, publish posts, upload avatars, and interact with other users through a dynamic feed.

The project demonstrates backend development, database management, and basic frontend integration.

---

## 🚀 Features

### 👤 User System
- Registration & authentication
- Profile page with:
  - avatar upload
  - bio (description)
  - username & nickname
  - location & join date

### 📝 Posts
- Create new posts
- Delete own posts
- View posts from other users
- Global feed (timeline)

### 🖼 Media
- Upload profile avatar
- Upload banner (profile header)

### 🌐 Pages
- Home (feed with posts)
- Profile page
- User profiles
